# malvinas.argentinadatos.com

Banners de las Islas Malvinas para uso libre, con variantes de tamaño y formato.

**Uso de las imágenes, variantes, URLs y ejemplos:** [https://malvinas.argentinadatos.com](https://malvinas.argentinadatos.com)

## Desarrollo rápido

```bash
pnpm install
pnpm run build:banners   # escribe public/*.png y *.jpg
pnpm run site:build      # sitio en site/dist/
pnpm run site:dev        # http://localhost:4321 (generá banners antes si hace falta)
```
